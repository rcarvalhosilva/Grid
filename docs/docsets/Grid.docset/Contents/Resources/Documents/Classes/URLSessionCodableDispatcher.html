URLSessionCodableDispatcher

public class URLSessionCodableDispatcher : NetworkCodableDispatcher

A URLSessionCodableDispatcher is a NetworkDispatcher that uses URLSession to dispatch a request. Made to be used with requests that expect a codable value as response
